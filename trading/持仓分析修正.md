## 创建飞书应用详细步骤

### Step 1：进入飞书开放平台
```
网址：https://open.feishu.cn/
用企业管理员账号登录
```

### Step 2：创建企业自建应用
```
1. 点击右上角「开发者后台」
2. 点击「创建企业自建应用」
3. 填写信息：
   - 应用名称：OpenClaw 数据助手
   - 应用描述：用于持仓数据同步和复盘报告生成
   - 应用图标：任意上传
4. 点击「确定创建」
```

### Step 3：添加权限
```
1. 左侧菜单点击「权限管理」
2. 搜索并添加以下权限：

   ✅ 获取用户基本信息
   contact:user.base:readonly
   
   ✅ 读取多维表格数据
   bitable:app:readonly
   
   ✅ 编辑多维表格数据（如需我写入）
   bitable:app:write
   
   ✅ 发送消息（如需推送）
   im:message:send_as_bot
   im:message.p2p_msg:write
```

### Step 4：获取凭证
```
1. 左侧菜单点击「凭证与基础信息」
2. 复制：
   - App ID（应用凭证）
   - App Secret（应用密钥）
3. 把这两个发给我
```

### Step 5：发布应用
```
1. 左侧菜单「版本管理与发布」
2. 点击「创建版本」
3. 填写版本号（如 1.0.0）和更新说明
4. 点击「保存」
5. 点击「申请发布」
6. 联系企业管理员审批通过
```

### Step 6：配置多维表格权限
```
1. 打开你的多维表格
2. 点击右上角「...」→「高级设置」→「添加应用」
3. 搜索「OpenClaw 数据助手」
4. 选择权限：可查看 / 可编辑
```
