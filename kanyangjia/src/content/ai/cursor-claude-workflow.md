## 🎯 是什么
Cursor 是 AI 原生的代码编辑器，配合 Claude 的大模型能力，可以极大提升编程和自动化效率。

## 💡 使用场景
- 快速搭建网站/应用原型
- 自动化脚本编写
- 代码重构和优化
- 学习新技术

## 🛠️ 操作步骤
1. 安装 Cursor (cursor.com)
2. 配置 Claude API Key
3. 用自然语言描述需求
4. AI 生成代码 → 审查 → 应用

## ✨ 效果展示
今天用 Cursor + Claude 完成了看养家网站的部署和修复，全程对话式编程，无需手写复杂命令。

## 💭 个人评价
**优点**：
- 无需记忆复杂命令
- 上下文理解能力强
- 可以直接执行终端命令

**缺点**：
- 需要审查 AI 生成的代码
- 复杂逻辑仍需人工把关

**适合人群**：想快速实现想法但不想深究技术细节的人。
